# Domain Service Endpoints

## Core Endpoints
- `GET /api/v1/domains` - List supported domains
- `POST /api/v1/domains/search` - Search across a specific domain (website, local, hiring)
- `POST /api/v1/sources/connect` - Connect external sources
- `POST /api/v1/sources/sync` - Sync data from an external source
- `GET /api/v1/signals` - List supported signal types
- `GET /api/v1/signals/{lead_id}` - Get signals for a specific lead
