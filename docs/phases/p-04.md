# 🔹 adapters/hiring_adapter.py

```python id="93mh4o"
class HiringAdapter:

            {
                "company_name": "Growth Labs",
                "signal": "hiring_backend_engineer"
            }
        ]
```

---

# 🔹 services/domain_service.py

```python id="7zjlwm"
class DomainService:

    def __init__(self):
        self.adapters = {
            "website": WebsiteAdapter(),
            "hiring": HiringAdapter()
        }

    def search(self, domain, keyword):
        adapter = self.adapters.get(domain)
        return adapter.search(keyword)
```
